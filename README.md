# FOOD VIEW 

## Description
A FULLSTACK application where we get reels of food and get the food from the food partner.

## Setup

### Backend
1. Go to `backend/` using Node.js and PostgreSQL
2. Install dependencies: `npm install`
3. Start server: `npm start`
4. Environment variables needed: `JWT_KEY`, etc.

### Frontend
1. Go to `frontend/` using React
2. Install dependencies: `npm install`
3. Start frontend: `npm start`

## Usage
Just create account and scroll the shorts( videos ) and get the desired food .
